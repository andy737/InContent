How To Use

In your HTML document, put the follow code for call the CSS: 

<link rel="stylesheet" href="css/incontent.css" type="text/css" >

Put the code below in your HTML document:

<div class="pic">
	<img src="img/01.jpg" class="pic-image" alt="Pic"/>
	<span class="pic-caption bottom-to-top">
		<h1 class="pic-title">Bottom to Top</h1>
		<p>Hi, this is a simple example =D</p>
	</span>
</div>
